# Daguerre
一个`Android`本地图片和视频的选择器
- 可以在Activity或Fragment中使用
- 选择图片支持多种格式，如：JPEG, PNG, GIF
- 选择视频支持多种格式，如：3GP, MPEG, MP4
- 可以自定义主题样式
- 可自定义不同的图片加载器
